# Todo List Application

This project is a simple **React.js** application that allows users to manage their tasks (todos) efficiently. It includes basic features such as adding, editing, and deleting tasks. Future updates will focus on improving the design and adding advanced features like a search algorithm for filtering tasks.

## Features

- Add new tasks.
- Edit existing tasks.
- Delete tasks.
- Dynamic interaction with tasks.
- Future plans include:
  - Adding a search bar to filter tasks.
  - Improving the design for better user experience.

## Technologies Used

- React.js for building the user interface.
- CSS for styling the application.
- React Icons for interactive icons like edit and delete.

## Installation

1. Clone the repository using:
   `git clone https://github.com/your-username/todo-list-app.git`
2. Navigate to the project directory:
   `cd TodoReact`
3. Install the required dependencies:
   `npm install`
4. Start the development server:
   `npm run dev`
5. Open your browser and navigate to:
   `http://localhost:5173/`

## Usage

When you open the application:
- You can add a task by typing in the input field and clicking the "Add Task" button.
- Tasks will appear in a list format. You can:
  - Edit a task by clicking the edit icon.
  - Delete a task by clicking the delete icon.

## Project Structure

The project is organized as follows:
- The `src` folder contains the main application components and styles.
- The `TodoList.jsx` component is responsible for managing the todo list functionality.
- The `App.jsx` component serves as the main entry point of the app.
- The `App.css` file contains the styles for the application.

## Future Enhancements

This project will be updated in the future with:
- A **search functionality** to allow users to filter tasks quickly and easily.
- Enhanced **design improvements** for a more polished and user-friendly interface.
- Additional features like sorting tasks by priority or deadline.
