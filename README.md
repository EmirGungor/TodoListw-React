# Todo List Application

This project is a simple **React.js** application that allows users to manage their tasks (todos) efficiently. It includes basic features such as adding, editing, and deleting tasks.

## Features

- Add new tasks.
- Edit existing tasks.
- Delete tasks.
- Dynamic interaction with tasks.

## Technologies Used

- **React.js**: To build the user interface.
- **CSS**: For styling the application.
- **React Icons**: For adding icons for edit and delete actions.
