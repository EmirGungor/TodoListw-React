# Todo List with React

A modern **Todo List** application built using **React.js** that helps users organize and manage their tasks efficiently. This app allows users to create, edit, delete, filter, and mark tasks as completed.

---

## Features
- **Add Tasks:** Users can easily add new tasks to their todo list.
- **Edit Tasks:** Existing tasks can be edited by clicking on the edit icon.
- **Delete Tasks:** Tasks can be removed with a single click.
- **Mark as Completed:** Users can mark tasks as completed using a checkbox. Completed tasks are visually distinguished with a strikethrough.
- **Filter Tasks:** A search bar allows users to filter tasks by keywords, making it easy to find specific tasks.
- **Responsive Design:** The app is designed to work seamlessly on desktops, tablets, and mobile devices.

## How to Use
1. **Add a Task:**
   - Type your task in the input field at the top and click the **"Add Task"** button.
2. **Edit a Task:**
   - Click the **edit icon** (🖉) next to the task, modify the text in the popup, and save your changes.
3. **Delete a Task:**
   - Click the **delete icon** (🗑️) to remove a task.
4. **Mark as Completed:**
   - Use the **checkbox** to mark a task as completed. Completed tasks will have a **strikethrough** style.
5. **Filter Tasks:**
   - Use the **filter input field** to search for tasks by keywords. Only matching tasks will be displayed.
